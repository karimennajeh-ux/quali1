# Documents Word de l'application

Ce dossier centralise les documents Word utilises par l'application.

## Structure

- `templates/`
  - modèles Word utilises par le module `Documentation`
- `reports/`
  - rapports Word generes ou conserves comme livrables

## Fichiers actuels

### Templates

- `modèle procédure.docx`
- `modèle instruction.docx`
- `modèle formulaire.docx`
- `modèle Enregistrement.docx`

### Reports

- `Rapport_soutenance_PFE_Quali_by_ENNAJEH.docx`

## Note

L'application principale reference maintenant les modèles depuis :

- `/database/word_documents/templates/...`

Le script de génération du rapport de soutenance est configure pour utiliser ce dossier comme emplacement cible.
